# Woontegra Optik Desktop

Optik/gözlükçü bayileri için masaüstü takip programı.

## Gereksinimler

- Node.js 18+
- Windows 10/11

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Windows Kurulum Paketi

```bash
npm run dist:win
```

## Demo Lisans

`WO-OPTIK-DEMO-2026`

## Veritabanı
